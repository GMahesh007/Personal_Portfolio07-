Software Network

  • Tech Stack: Visual.Net, MySQL Server
  • GitHub URL: Project Link

Aim

The main objective of the project is to simplify the management of software development projects and track bugs efficiently. It helps automate several tasks like ticket creation, bug assignment, and status tracking, thus improving team communication and productivity.

Modules

  1. Home Module: Features home page, contact form, and login/registration for various roles (Admin, Manager, Team Leader, Developer).
  2. Admin Module: Allows the admin to manage user roles, update project details, and view feedback.
  3. Manager Module: Enables project creation in IEEE format, approves developer registrations, and tracks assigned bugs.
  4. Team Leader Module: Facilitates bug ticket creation, bug assignment to developers, status tracking, and communication via chat system.
  5. Developer Module: Allows developers to update bug statuses, view assigned bugs, and communicate with team leaders through the chat system.

Conclusion

In conclusion, the Software Network project is a powerful tool for managing software development tasks and bug tracking. By automating key processes, it enhances communication between the team and ensures a more efficient workflow, ultimately leading to faster project delivery.
